# BRAVOXX - Projeto de Briefing e Protótipo

## Sobre o projeto

Este projeto foi desenvolvido como atividade acadêmica com o objetivo de criar um **briefing completo** e um **protótipo de site** para uma marca fictícia de roupas no estilo streetwear.

A proposta foi construir uma identidade visual moderna, impactante e alinhada ao público consumidor de moda urbana.

---

##  Objetivo

Desenvolver um site voltado para **venda de roupas**, com foco em:

* Identidade visual forte
* Estilo urbano (streetwear)
* Experiência visual impactante
* Organização para conversão de vendas

---

## Público-alvo

Consumidores de moda interessados em:

* Estilo urbano
* Tendências atuais
* Peças exclusivas

## Conceito do projeto

A marca **BRAVOXX** foi pensada com base em uma estética **urbana e agressiva**, inspirada em elementos como:

* Grafite
* Street art
* Cultura streetwear

O design utiliza alto contraste (preto + verde neon) para gerar impacto visual e chamar atenção do usuário.
